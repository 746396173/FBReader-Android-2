Latest official version of FBReader for Android 2.* (2.6.12) published on 31 March 2016. There will be no update longer.

The repo contains complete sources of this latest release for standard Android 2 devices (branch master) and for Nook Simple Touch (branch nook).

It uses gradle for build.
